Basic JavsScript Syntax and Rules

Javascript is case sensitive

alert() is not same as Alert()

Similarly, myFunction() is not same as myfunction()

Each statement should be ended with semi-colon ";" , in this case Js is not sensitive, but it is good practice if we end each statement with '';'

Summary

  • JavaScript is case sensitive
  • Contains many statements, all ending with a semi-colon ;
  • Nott sensitive to line breaks or whitespace
  • Write one line comment using //
  • Write multi line comment using /* *\
  • JavaScript runs from top to bottom
