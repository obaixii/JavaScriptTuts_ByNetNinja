What is the DOM

The DOM is an application programming interface

We use the DOM when we interact with web pages. We Can :

  • Adds content to a HTML document
  • Deletes content from a HTMl document
  • Change/Modify content on a HTML document

Every HTMl element is the document is an object

  • Head is and object
  • Body tag is an object
  • Every tag or element inside HTMl is an object

Everything we Change in the HTML document is called or known as node

  • Elements
  • Text within element
  • HTML attributes
